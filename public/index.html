    Local AI model not detected
    
          
    This feature uses the LanguageModel API — a local model (Gemini Nano / Phi-mini) that runs entirely in your browser. No API keys needed.
    
          
    
          
    Google Chrome (127+)
    
          
      
            
    1. Open chrome://flags
      2. 
            
    2. Set #prompt-api-for-gemini-nano → Enabled
      3. 
            
    3. Set #optimization-guide-on-device-model → Enabled BypassPerfRequirement
      4. 
            
    4. Navigate to chrome://restart
      5. 
            
    5. Go to chrome://components → find Optimization Guide On Device Model → click Check for update
      6. 
            
    6. Wait for download (~3-4 GB), then refresh this page
      7. 
          
    
          
    Microsoft Edge (Dev/Canary)
    
          
      
            
    1. Open edge://flags
      2. 
            
    2. Set Prompt API for Phi mini → Enabled
      3. 
            
    3. Set #optimization-guide-on-device-model → Enabled BypassPerfRequirement
      4. 
            
    4. Navigate to edge://restart (kill Edge background processes if needed)
      5. 
            
    5. Go to edge://components → find Optimization Guide On Device Model → click Check for update
      6. 
            
    6. Wait for download, then refresh this page
      7. 
          
    
          
    Verify in DevTools (F12)
